## 💡 Советы

### Если что-то не так
1. Перезагрузите приложение (встряхните телефон и нажмите "Reload")
2. Очистите кэш: `npm start -- --clear`
3. Перезагрузите Expo Go на телефоне

### Для лучшего опыта
1. Используйте LAN режим: `npm start -- --lan`
2. Держите телефон рядом с компьютером
3. Используйте хороший Wi-Fi

---

## 📞 Если есть проблемы

1. Проверьте консоль в терминале на ошибки
2. Убедитесь, что все зависимости установлены: `npm install`
3. Очистите кэш: `npm start -- --clear`
4. Перезагрузите Expo Go на телефоне
