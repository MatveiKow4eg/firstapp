- **Анимации**: Масштабирование при нажатии

- **Новые размеры**: xs (11px), xxl (24px)
- **Лучшая иерархия**: Заголовки, подзаголовки, текст

## 🔘 Компоненты

### PrimaryButton
```typescript
// Primary (по умолчанию)
<PrimaryButton title="Сохранить" onPress={handleSave} />

// Secondary
<PrimaryButton 
  title="Отмена" 
  onPress={handleCancel}
  variant="secondary"
/>

// Danger
<PrimaryButton 
  title="Удалить" 
  onPress={handleDelete}
  variant="danger"
/>
```

### AgreementCard
```typescript
<AgreementCard
  agreement={agreement}
  onPress={() => navigation.navigate('AgreementEdit', { id: agreement.id })}
  onLongPressToggleDone={(id) => setStatus(id, 'done')}
/>
```

### EmptyState
```typescript
<EmptyState message="Нет договорённостей" />
```

## 💡 Советы для разработчиков

### Используйте тему
```typescript
import { theme } from './src/theme/theme';

// ✅ Правильно
backgroundColor: theme.colors.primary

// ❌ Неправильно
backgroundColor: '#6366F1'
```

### Используйте компоненты
```typescript
// ✅ Правильно
<PrimaryButton title="Сохранить" onPress={handleSave} />

// ❌ Неправильно
<Pressable onPress={handleSave}>
  <Text>Сохранить</Text>
</Pressable>
```

### Используйте отступы из темы
```typescript
// ✅ Правильно
padding: theme.spacing.lg

// ❌ Неправильно
padding: 16
```
